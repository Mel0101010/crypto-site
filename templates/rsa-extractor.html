À propos de l'extraction RSA

RSA est un algorithme de chiffrement à clé publique basé sur la difficulté de factoriser le produit de deux grands nombres premiers. Si ces nombres premiers peuvent être retrouvés, la clé privée peut être reconstruite à partir de la clé publique.

Méthodes d'analyse implémentées:

  • Facteurs communs - Vérifie les facteurs premiers connus et faibles
  • Méthode de Fermat - Efficace lorsque les facteurs sont proches l'un de l'autre
  • Vulnérabilité d'exposant faible - Détecte les problèmes avec les petits exposants publics
  • Nombres premiers partagés - Vérifie si le modulus partage un facteur avec une base de données de nombres premiers connus
